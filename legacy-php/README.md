# Versi PHP + MySQL (XAMPP)

Versi server lama. Untuk menjalankan:

1. Import `database/schema.sql` ke MySQL
2. Sesuaikan `config/database.php`
3. Akses `http://localhost/melianawebsitekamera/legacy-php/index.php`

Versi utama proyek sekarang **static HTML** di root (`index.html`) untuk GitHub Pages.
